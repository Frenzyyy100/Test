📋 Changelog

What's Changed

Every fix, feature, and update to NovéLore — tracked and documented.

v0.6.9 — Current

🗓 April 2026 · Beta
✨ Reader Preferences panel — tap "Aa" in the chapter nav to adjust settings mid-read
✨ Font size, line spacing, and reading width now adjustable without leaving the chapter
✨ Theme switcher (Light / Dark / System) accessible directly from the reader
✨ Auto-Mark on Finish and Continue Where You Left Off toggles in reader panel
⚙️ Preference controls now stay in sync between Reader panel and Settings screen

v0.6.8 — Previous

🗓 April 2026 · Beta
🗑️ Removed Dojo Duels tab from the home screen
✨ Book Updates section added to the Updates tab
✨ Volume Updates section added to the Updates tab
⚙️ Updates tab now shows Book → Volume → Chapter in order

v0.6.7 — Previous

🗓 April 2026 · Beta
✨ Changelog page now fully respects your theme — Light, Dark, and System modes all apply correctly
⚙️ Changelog hero banner is now theme-aware (removed hardcoded dark gradient)
⚙️ About hero banner now adapts to Light Mode for consistent theming across all pages

v0.6.6 — Previous

🗓 April 2026 · Beta
⚙️ Removed "Clear all" button from notifications drawer
⚙️ Removed "Clear all" button from bookmarks drawer
✨ Bookmarks now persist — saved and restored across visits
✨ Notifications now persist — read/cleared state saved and restored across visits
🐛 Fixed end screen looping back when clicking the back button after finishing a chapter
🐛 Fixed NEW tag showing on last 3 chapters — now only shows on the latest chapter
🐛 Fixed chapter scroll not resetting to top when clicking Prev / Next
🐛 Fixed nav strip causing scroll anchor issues — moved outside scroll container
🐛 Fixed duplicate coverImg syntax error in bookmark logic
✨ Added cover image for The Lantern Pact: Prologue across all views
✨ Added The Lantern Pact to Hot Picks tab
✨ Added Changelog page (you're reading it!)
⚙️ All tab and panel switches now snap instantly to top
⚙️ Cleared default bookmark data

v0.5.0 — Reading Experience

🗓 March 2026 · Beta
✨ Chapter reader with per-chapter scroll progress tracking
✨ Auto-mark chapter as read at 95% scroll
✨ End-of-story screen with random messages
✨ Reading progress bar at top of chapter
✨ Prev / Next chapter navigation strip
⚙️ Continue Where You Left Off feature added to settings

v0.4.0 — Library & Discovery

🗓 February 2026 · Alpha
✨ Trending, Hot Picks, Frenzyyy's Work tabs
✨ What's New section with novel cards
✨ Chapter updates feed
✨ Announcements panel
✨ Hero slider with featured novels
✨ Novel detail page with synopsis, genres, and volume list

v0.3.0 — Bookmarks & Notifications

🗓 January 2026 · Alpha
✨ Bookmark system with mini popup and full drawer
✨ Notification system with unread badges
✨ Mark all as read / clear all notifications
⚙️ Sidebar navigation added (burger menu)

v0.2.0 — Settings & Themes

🗓 January 2026 · Alpha
✨ Dark / Light / System theme support
✨ Font size, line spacing, reading width controls
✨ Reduce motion and high contrast toggles
✨ Settings screen with reading preferences
✨ About page

v0.1.0 — Foundation

🗓 January 2026 · Alpha
✨ Initial build of NovéLore
