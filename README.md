## 📋 Overview

NestIQ is a cloud-based hotel management system built for small to medium-sized independent hotels and resorts across the Philippines (10–120 rooms). It replaces spreadsheets, group chats, and paper logbooks with one clean, affordable platform.

**Target Market:** Independent hotels in tourism hubs — Palawan, Boracay, Siargao, Bohol, Cebu, and urban areas like Makati and BGC.

## 🚀 Features

- **Front Desk & Reservations** — Real-time room availability, check-in/check-out, no double bookings
- **Housekeeping Tracker** — Assign rooms, track cleaning status, log damages
- **QR Guest Requests** — Guests scan a QR code to submit requests (no app needed)
- **Owner Dashboard** — Daily occupancy, revenue, and issues at a glance
- **GCash & Maya Payments** — Built-in Philippine digital payment support
- **Filipino Language Interface** — English/Filipino toggle for staff

## 🛠️ How to Run Locally

No build tools needed — this is a pure HTML/CSS/JS project.

```bash
git clone https://github.com/your-username/nestiq.git
cd nestiq
# Open index.html in your browser
open index.html
```

Or use a simple local server:

```bash
npx serve .
# or
python -m http.server 8000
```
